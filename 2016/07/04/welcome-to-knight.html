Welcome to Knight

Jul 4, 2016 • Nick O'Brien


Everybody has an idea. Everybody has a passion of some sort. Everybody has things they care about.

This is where I’m going to cast my idea’s, drive my passions, and project what I care about.

Well, now that we’re done with the deep quote, welcome! This is my first big web project and I’m really proud of it. This was my first time using jekyll and I think it’s gone well. Along with Jekyll; this site uses Boostrap 3, a front-end styling frameworking using JavaScript and CSS. I also use Github’s free web hosting to host this site. I’ve had a lot of fun making this site. If you or anybody you would like to help writing posts I have some of my contact info below and on obrien.tech.

Have fun poking around!
